# Accessing Strings

You need to make a program that counts the number of vowels in a given text. The vowels are **a**, **e**, **i**, **o**, and **u**.

Take a string as input and output the number of vowels.

---

**Sample input**: 
```
this is great
```

**Sample output**: 
```
4
```

---

>There are 4 vowels in the given text.
