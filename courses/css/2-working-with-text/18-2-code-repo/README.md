# Text Transform

Moving ahead! In this stage, we will make the heading and subheading text appear in all-uppercase.

**Task**: In the CSS code, make the `heading` and `subheading` sections uppercase using the `text-transform` property.

**Example**
```css
.heading {
    font-size: 35px;
    font-weight: bolder;
    text-transform: uppercase;
} 

.subheading {
    font-size: 20px;
    font-weight: bolder;
    color: #2F5A78;
    text-transform: uppercase;
}
```
