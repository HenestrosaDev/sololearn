# Dictionaries

You are working at a car dealership and store the car data in a dictionary:
```
car = {
	'brand': 'BMW',
	'year': 2018,
	'color': 'red'
} 
```

Your program needs to take the key as input and output the corresponding value.

---

**Sample input**: 
```
year
```

**Sample output**: 
```
2018
```

---

>The data is already defined in the code.
